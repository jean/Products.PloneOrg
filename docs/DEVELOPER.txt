
Development
-----------

Base
~~~~

A base build, suitable for testing the ``Products.PloneOrg`` add-on is described
in the top-level README.txt of this package.

Development
~~~~~~~~~~~

A development build, suitable for developing the latest 

    $ python2.6 bootstrap.py
    $ bin/buildout -c production.cfg
    $ bin/instance fg

Production
~~~~~~~~~~

A production build, suitable for running plone.org can be built with::

    $ python2.6 bootstrap.py
    $ bin/buildout -c production.cfg
    $ bin/supervisord

Database
~~~~~~~~

If you need the Data.fs and CatalogData.fs files from plone.org, and you have 
a plone.org UNIX system account you can do this::

    $ python2.6 bootstrap.py
    $ bin/buildout -c database.cfg
    $ bin/instance fg

If you do not have an account but want one, please contact the admins team
(admins@plone.org).
