PloneOrg buildout
=================

This is the plone.org site package and corresponding buildout. It is used to
power http://plone.org.

Install
-------

To install and run Plone (only), follow these steps::

    $ python2.6 bootstrap.py -d
    $ bin/buildout
    $ bin/instance fg
